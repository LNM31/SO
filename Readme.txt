Programul este implementat in treasure_manager.c cu executorul treasure_manager.

Exemple de executare a fiecarei operatiuni:

-ADD: ./treasure_manager add Hunt001 [urmand a fi introduse datele pentru comoara din hunt-ul Hunt001]
-LIST: ./treasure_manager list Hunt002
-VIEW: ./treasure_manager Hunt003 2 [id=2]
-REMOVE_TREASURE: ./treasure_manager remove_treasure Hunt004 3 [id=3]
-REMOVE_HUNT: ./treasure_manager Hunt005
